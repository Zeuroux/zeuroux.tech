Launchly

An open-source Minecraft®: Bedrock Edition version switcher.
Note: you have to buy Minecraft from the Google Play Store in order to use this, we do not support piracy.
This is not a piracy app.



Features

  • Version Switching
  • Managing Downloaded Versions (customizing icons, names, etc.)
  • Making Modifications (Soon™)

    • Modifications (Soon™):
  • Force External Storage
  • MaterialBinLoader2
  • AndroidModLoader
